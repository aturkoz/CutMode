# CutMode

> The easiest way to split 3D models in Blender for 3D printing.

![CutMode Banner](images/cutmode.png)

## Features

- ✂️ Cut any model anywhere
- 🟢 Automatically add Snap Connectors
- 🔧 Manual connector placement
- 📏 Adjustable connector size
- 📐 Adjustable clearance
- 🔢 Multiple connectors
- 📦 Export directly to STL
- ↩️ Undo last cut
- 🖨️ Designed for 3D Printing

---

## Installation

1. Download **CutMode.zip** from the **Releases** page.
2. Open Blender.
3. Go to **Edit → Preferences → Add-ons**.
4. Click **Install...**
5. Select **CutMode.zip**
6. Enable **CutMode**
